## üö¢ Project 1: Titanic Survival Analysis  

This Tableau workbook explores the **Titanic Dataset**, focusing on understanding passenger demographics, fares, survival patterns, and relationships between various factors that influenced survival outcomes.  

### üìå Dataset Description  
The Titanic dataset contains detailed information about passengers aboard the RMS Titanic, including:  
- **Gender, Age, Passenger Class, Fare**  
- **Survival Status (Survived / Not Survived)**  
- **Embarkation Port**  
- **Family Relations (SibSp, Parch)**  

This dataset is widely used for analytics and machine learning experiments due to its historical and well-structured nature.  

### üìä Visualization Tasks  

| No. | Task |
|-----|------|
| 1 | Bar chart of **Passenger Count vs. Gender** |
| 2 | **Survival Count by Gender** |
| 3 | **Survival Rate by Passenger Class** |
| 4 | **Age Distribution** (Histogram / Density) |
| 5 | **Average Fare by Class** |
| 6 | Scatter Plot of **Fare vs Age** |
| 7 | **Passenger Count by Embarkation Port** |
| 8 | **Survival Heatmap** based on Gender and Class |
| 9 | **Box Plot of Age vs Survival** |
| 10 | **Pie Chart** showing % of Total Passengers by Gender (or other dimensions) |

### üõ†Ô∏è How to Use

2. Open it using **Tableau Desktop** or **Tableau Public (installed app)**.

4. Optional dashboards or story pages summarize the findings.

## ‚òï Project 2: Coffee Chain Analytics  

This Tableau workbook is based on the **Coffee Chain Dataset**, focusing on sales, profit, marketing, and regional performance. The dataset is commonly used in Tableau learning modules to practice data analytics and visualization.  

### üìå Dataset Description  
The dataset includes information about:  
- **Product Type & Product Name**  
- **Sales, Profit, Expenses, and Marketing**  
- **State, Market, Area Code, and Region**  
- **Order Year and Month**  

### üìä Objectives & Tableau Tasks  

| No. | Task Description |
|-----|------------------|
| (i) | Create a **Heat Map** for Product Type, State, and Profit. Identify which state has the lowest profit for **Espresso**. |
| (ii) | Using a **Boxplot**, identify which Espresso product has the **highest distribution of Sales**. |
| (iii) | Create a **Bar Chart** with Product Type, Product, and Profit. Write conclusions. |
| (iv) | Create a **Scatter Plot** with State, Sales, and Profit. Write observations. |
| (v) | Identify the **state with the highest and lowest profit** in the market. |
| (vi) | Find the **number of Sales in Nevada for the product _Decaf Espresso_**. |
| (vii) | Calculate the **contribution of Tea (in %) to total Profit**. |
| (viii) | Find the **average Marketing spent** in Area Codes **660 and 818**. |
| (ix) | In **California**, identify the **product with highest and lowest profit**. |
| (x) | Using **Side-by-Side Circles**, find the **minimum marketing value** for **Coffee Beans Colombian**. |
| (xi) | Find the **contribution of Sales in East Market for Decaf**. |
| (xii) | In **2012**, find the **contribution of Sales in East Market for Decaf**. |
| (xiii) | Calculate the **average Profit of all products starting with letter "C"**. |
| (xiv) | In **Central Region**, find what % of total sales is contributed by **Top 5 products by sales**. |
| (xv) | In **2013**, identify the **state with the highest profit in West Market**. |
| (xvi) | Find the **Total Expenses to Sales Ratio** for the **state with the lowest profit**. |
| (xvii) | Find the **distinct count of Area Codes** for the **state with lowest Budget Margin in Small Markets**. |
| (xviii) | In **2013**, find the **% of total Profit for Caffe Mocha under Major Market**. |
| (xix) | Identify the **Month & Year when Sales crossed $1100** for **Decaf in Colorado and Florida**. |
| (xx) | Using **Treemaps**, identify the **Market with maximum Budget Sales**. |

### üõ†Ô∏è How to Use  
1. Download the **`Coffee_Chain_Analytics.twbx`** file from this repository.  
2. Open it in **Tableau Desktop** or **Tableau Public (installed version)**.  
3. Navigate through each worksheet to explore visuals corresponding to the listed questions.  
4. Dashboards and Story pages summarize major insights and KPIs.
